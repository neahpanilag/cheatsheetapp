<a> @ [HTML] Defines a hyperlink @ <h2>Definition and Usage</h2><br><p>The &lt;a&gt; tag defines a hyperlink, which is used to link from one page to another.</p><br><p>The most important attribute of the &lt;a&gt; element is the href attribute, which indicates the link's destination.</p><br><p>By default, links will appear as follows in all browsers:</p><br><ul><li>An unvisited link is underlined and blue</li><li>A visited link is underlined and purple</li><li>An active link is underlined and red</li></ul>

<caption> @ [HTML] Defines a table caption @ <h2>Definition and Usage</h2><br><p>The &lt;caption&gt; tag defines a table caption.</p><br><p>The &lt;caption&gt; tag must be inserted immediately after the &lt;table&gt; tag.</p><p><strong>Note:</strong> You can specify only one caption per table.</p><p><b>Tip:</b> By default, a table caption will be center-aligned above a table. However, the CSS properties <em>text-align</em> and <em>caption-side</em> can be used to align and place the caption.</p>

The &lt;img&gt; tag creates a holding space for the referenced image.</p><br><p><b>Tip:</b> To link an image to another document, simply nest the &lt;img&gt; tag inside &lt;a&gt; tags.</p>
